# Q1. 막대기

## 문제
높이가 서로 다른 $N$개의 막대기가 일직선으로 세워져 있다. 이 막대기들을 오른쪽에서 보았을 때, 보이는 막대기의 개수를 구하는 프로그램을 작성하시오.

오른쪽에서 보았을 때 어떤 막대기가 보이려면, 그 막대기보다 오른쪽에 있는 모든 막대기들보다 높이가 높아야 한다.

## 입력
첫째 줄에 막대기의 개수 $N$이 주어진다.

다음 $N$개의 줄에 걸쳐 각 막대기의 높이 $h_i$가 왼쪽부터 순서대로 주어진다.

* $1 \le N \le 100,000$
* $1 \le h_i \le 100,000$

## 출력
오른쪽에서 보았을 때 보이는 막대기의 개수를 출력한다.

## 입출력 예시

**예시 1**
* 입력: 

$6$

$6$

$9$

$7$

$6$

$4$

$6$

* 출력: 

$3$

**예시 2**
* 입력: 

$5$

$5$

$4$

$3$

$2$

$1$

* 출력: 

$5$

```python
## Q1. 막대기
# -*- coding: utf-8 -*-
# UTF-8 encoding when using korean
N = int(input())
count,h_max, h = 0, 0, []
for i in range(N):
    h.append(int(input()))
h_max_index = h.index(max(h))

for i in range(h_max_index,len(h))[::-1]:
    if h[i] > h_max:
        count+=1
        h_max = h[i]
print(count)

```

## 풀이과정
1. 배열을 역순으로 배치 해야하니 `range()`함수에 `[::-1]`을 사용
2. `h.index(max(h))`으로 불필요한 서치를 제한
3. 뒤에서부터 비교하여 큰 숫자가 나올때까지 비교, 개수를 카운트
# Q2. 소금물의 농도 구하기

## 문제
$7\%$의 소금물 $Ng$이 있다. 이 소금물에 $Mg$만큼의 물을 넣었을 때, 소금물의 농도를 소수 $2$번째 자리까지 표현하고, 나머지는 버림하여 출력하시오

## 입력
첫째 줄에 소금물의 양 $N$과 소금의 양 $M$이 공백을 두고 주어진다.

* $10 \le N \le 10,000$
* $1 \le M \le N$

## 출력
소금물의 농도를 소수점 세 번째 자리에서 버림하여 소수점 두 번째 자리까지 출력한다.

## 입출력 예시

**예시 1**
* 입력: 

$1000 \ 0100$

* 출력: 

$3.50$

**예시 2**
* 입력: 

$100 \ 150$

* 출력: 

$2.80$

```python
# Q2. 소금물의 농도 구하기
# -*- coding: utf-8 -*-
# UTF-8 encoding when using korean
# 기본 7%, N = 7% 소금물 Ng, M = 물 Mg
N, M = map(int, input().split())
salt = N * 0.07
print(f"{int(((salt/(M+N))*100)*100)/100:.2f}")
```

## 풀이과정
소금물 공식 (소금 양 / 전체 양) *100

소수 끝자리가 버림이므로

100을 곱핸 다음 int형으로 형변환(버림),

다시 100으로 나누어 출력
