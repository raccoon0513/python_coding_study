# Q2. 의좋은 형제

## 문제
조선 시대에 수도 한양에서 한참 떨어진 곳에 있는 한 고을에는 의좋은 형제가 살았다. 서로를 끔찍이 아낀다고 소문난 구름이와 바람이 형제는 항상 서로에게 퍼주지 못해 안달이 난 상태였다. 의리 있는 만큼 성실했던 두 형제는 열심히 일하여 번 돈으로 식량을 어마어마하게 비축하여 가지고 있었다.

그러던 어느 날, 구름이는 밤 중에 몰래 집을 나와 자신의 식량 절반을 바람이에게 가져다주었다. 이 사실을 안 바람이는 다음 날 밤, 몰래 집을 나와 마찬가지로 자신의 식량 절반을 구름이에게 가져다 주었다. 가지고 있는 식량의 양이 홀수라서 반으로 나눌 수 없는 경우, 그 식량을 통째로 넘겨준다.

매일 밤 계속되는 이 모습을 지켜본 동네 꼬마는 문득 $D$번째 날에는 두 형제에게 식량이 얼마나 남아있을지 궁금해졌다. D번째 날에 구름이와 바람이는 식량을 얼마나 갖고 있을지 출력하시오.



## 예제 설명

첫 번째 예제에서 구름이와 바람이는 모두 처음에 식량을 **100**만큼 가지고 있다.

1. 첫째 날에는 구름이가 바람이에게 식량을 **50**만큼 준다. 이후 구름이는 식량을 **50**, 바람이는 식량을 **150**만큼 가지고 있다.
2. 둘째 날에는 바람이가 구름이에게 식량을 **75**만큼 준다. 이후 구름이는 식량을 **125**, 바람이는 식량을 **75**만큼 가지고 있다. 
3. 셋째 날에는 구름이가 바람이에게 식량을 **63**만큼 준다. 이후 구름이는 식량을 **62**, 바람이는 식량을 **138**만큼 가지고 있다.
4. 넷째 날에는 바람이가 구름이에게 식량을 **69**만큼 준다. 이후 구름이는 식량을 **131**, 바람이는 식량을 **69**만큼 가지고 있다. 

따라서 넷째 날이 지난 이후 구름이와 바람이가 가지고 있는 식량의 양은 순서대로 **131**, **69**이다.



## 입력

첫째 줄에는 구름이와 바람이가 처음 갖고 있는 식량의 양 $N_1, N_2$ 가 공백을 두고 주어진다.
둘째 줄에는 두 형제의 식량의 양을 알고 싶은 날 $D$가 주어진다.

- $1 \leq N_1, N_2 \leq 10,000$
- $1 \leq D \leq 100$
- 입력에서 주어지는 수는 모두 정수이다.


## 출력

$D$번째 날에 구름이와 바람이가 가지고 있는 식량의 양을 순서대로 공백을 두고 출력한다. 

입력값 :
```
100 100
4
```

목표 출력값 : `131 69`


```python
# Q2. 의좋은 형제
# -*- coding: utf-8 -*-
# UTF-8 encoding when using korean

class bro:
    def __init__(self, food):
        self.food = food

    def give_cal(self, count, other):
        self.food = self.food - count
        other.take(count)

    def give(self, other):
        if(self.food % 2):
            self.give_cal(self.food//2+1, other)
        else:
            self.give_cal(int(self.food/2), other)

    def take(self, count):
        self.food = self.food + count


N1, N2 = map(int ,input().split())
D = int(input())
bros = []
bros.append(bro(N1)) # 얘가 먼저 줌
bros.append(bro(N2))

for i in range(D):
    bros[i%2].give(bros[(i+1)%2])

print(bros[0].food, bros[1].food)
#출력은 bro0, bro1 순
```

## 풀이과정
클래스를 공부해보고 싶어서

형제를 클래스로 정의했다

`give_cal()` 함수의 이름을 원래 `give`로 하려 했으나

`take`함수의 통일성을 위해 `give_cal`로 변경하고

조건 체커의 이름을 `give()`로 설정했다

문제의 이해를 잘못해서 주는 식량이 홀수면

모든 식량을 넘겨주는 줄 알았다

그 반절된 식량 하나만 넘겨주는것이었기에

`self.give_cal(self.food, other)`을

`self.give_cal(self.food//2+1, other)`로 수정했다

매 턴 번갈아가면서 식량을 주게 하기 위해

`bros[i%2].give(bros[(i+1)%2])`를 사용했다

# Q7. 합 계산기

## 문제

보통의 계산기는 한 번에 하나의 계산 결과만 나타낼 수 있다. 그래서 여러 개의 계산 결과가 필요한 경우에는 이전 계산 결과를 따로 기록해 둬야 하는 번거로움이 있었다.

플레이어는 이러한 점을 해결하기 위해서 **합 계산기**를 만들었다. 합 계산기는 여러 개의 계산식을 입력받은 뒤, 각각의 계산 결과를 모두 합해서 출력하는 기능을 가지고 있다. 합 계산기에 입력할 수 있는 계산식은 아래 조건을 만족해야 한다.

- `<연산 기호>` 에는 더하기, 빼기, 곱하기, 나누기의 네 가지 사칙 연산 기호가 들어갈 수 있다. 이때, 나눗셈 결과의 나머지는 버린다.

합 계산기에 입력할 $T$개의 계산식이 주어질 때, 합 계산기의 출력 결과를 구해보자.

다음 $T$개의 줄에는 계산식이 주어진다.

- 계산식은 `<정수> <연산 기호> <정수>` 형태로 주어진다.
- `<정수>` 에는  $1$이상  $1,000$이하의 정수가 주어진다.
- `<연산 기호>` 에는 `+`, `-`, `*`, `/` 중 하나의 문자가 주어진다. 각각 더하기, 빼기, 곱하기, 나누기 기호를 의미한다.

## 출력

합 계산기가 계산한 결과를 출력하시오.

입력값 : 
```
3
1 + 3
4 / 3
3 - 2
```

목표 출력값 : 
```
6
```

입력값 : 
```
3
3 - 4
4 / 5
5 * 1
```

목표 출력값 : 
```
4
```



```python
# Q7. 합 계산기
# -*- coding: utf-8 -*-
# UTF-8 encoding when using korean
def cul(a, math_sign, b):
    a,b =int(a),int(b)
    if math_sign == "+":
        return a + b
    if math_sign == "-":
        return a - b
    if math_sign == "/":
        return a // b
    if math_sign == "*":
        return a * b

T = int(input())
result = 0
for i in range(T):
    a, math_sign, b = input().split()
    result+= cul(a, math_sign, b)
print(result)
```

## 풀이과정

를 `lambda`함수를 써서 대체해보고 싶다

그래서 나온 것이
```python
def time_normalrization(T, M):
    return (lambda now: (list(map(str, [(now//60)%24,now%60]))))(T * 60 + M)
```
기괴한 형태가 나왔다
1. 분은 60으로 나눈값의 나머지
2. 시는 60으로 나눈 몫
3. 시가 24를 초과 할 수 있으니 24로 나눈 값의 나머지를 썼다
4. 문자열 결합을 위해 매핑, str로 형변환
5. list객체로 반환
6. 함수 밖에서 `" ".join()`을 통해 합체, 출력

## 문제

근력 운동을 할 때, 1회에 최대한으로 들 수 있는 무게를  $1RM$이라고 한다. 본인의 $1RM$이 얼마나 되는지를 알아야 효율적인 운동 방식을 고를 수 있어 $1RM$을 측정하는 것은 무척 중요하다. 그러나 무작정 무거운 무게를 들어서 측정하는 방식은 다칠 위험이 크므로, 보통은 다양한 공식을 사용해서 $1RM$을 추정한다. 최대 무게가 아닌 적당한 무게를 몇 번 반복해 들었나를 가지고 내가 한 번에 들 수 있는 최대 무게를 추정하는 식이다. 이번 문제에서는 아래와 같은 공식을 사용한다. 는 $W$무게, $R$은 반복 횟수를 의미한다.

첫째 줄에 플레이어가 스쿼트 한 세트에서 들어올린 무게 $W$와 반복 횟수 $R$이 공백을 두고 주어진다.


- $1 \leq W \leq 10^6$
- $1 \leq R \leq 100$
- 입력에서 주어지는 모든 수는 정수이다.

## 출력

플레이어의 $1RM$을 소수점 이하의 값은 버린 뒤 출력한다.

```
100 30

```
