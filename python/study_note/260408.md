# Q1. 직각 만들기 1
## 문제

직각을 매우 좋아하는 구름이는 평면에 놓인 $N$개의 점에 대해 임의의 3개의 점을 순서대로 이은 도형이 직각을 만들 수 있는 경우의 수를 구하고자 한다. 점의 순서가 다를 경우 서로 다른 경우로 간주한다. 구름이를 도와 직각을 만들 수 있는 경우의 수를 계산하라.

## 입력

첫째 줄에 점의 개수 $N$이 주어진다.
다음 개의 줄에 점의 좌표 $X_i$, $Y_i$가 공백을 두고 주어진다.
 - $3 \leq N \leq 1500$
 - $-10^9 \leq X_i, Y_i \leq 10^9; 1 \leq i \leq N$
 - 주어지는 점의 좌표는 모두 다르다.



## 출력

첫째 줄에 직각을 만들 수 있는 경우의 수를 출력한다.

예제 문제
4

1 0
